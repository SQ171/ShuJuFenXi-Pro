### 1. 项目初始化
- 建立 Git 仓库 → GitHub 远程仓库连接
- 目录结构：core / parsers / analysis / visualization / ui / export

### 2. 核心功能实现

| 模块 | 文件 | 功能 |
|---|---|---|
| 数据解析 | `parsers/metv6.py` | MET-V6 CSV 解析，5文件 ~52万行 |
| 分析引擎 | `analysis/` | Pipeline 8步骤：加载→台阶检测→运行时间→稳态统计→特征提取→退化分析→异常检测→多文件对比 |
| 特征工程 | `analysis/features.py` | 8时域特征(RMS/CV/偏度/峰度等) + 7频域特征(FFT频谱分析) |
| 可视化 | `visualization/` | 时序图/台阶箱线图/退化趋势/FFT对比/健康雷达图 |
| UI | `ui/pages/` | 7页面：概览/时序探索/台阶分析/特征分析/多文件对比/退化趋势/报告导出 |
| 导出 | `export/excel_exporter.py` | Excel 5 Sheet |

### 3. 分析结果摘要
- 总运行时间：14.39h，1035个循环，10个油门台阶(20-65%)
- 力效 @40%油门：修正后 ~4.6 g/W（标准大气条件下）
- 电机状态良好，五维健康评分均 >0.98

### 4. 关键修复
- Sigma 滑块失效 → 通过 pipeline 传递
- 退化率单位错误（×3600） → 改用实际循环时长
- 力效大气修正：等效因子 = √(ρ_std/ρ_actual)
- 健康雷达评分：从拍脑袋公式 → 基于满血/报废阈值的线性映射
